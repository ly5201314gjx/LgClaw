# Summarize

Summarize URLs, local files, and YouTube links.

## Trigger Phrases

Use when user asks:
- summarize this URL/article/video
- what's this link/video about
- transcribe this YouTube/video

## Notes

- If transcript is huge, return concise summary first.
- Offer expansion by section/time range.
- Prefer machine-readable output only when user asks.
